Ready?

Draw basic shapes (1/5)

Draw the basic shapes used as building blocks for the camera icon. Use the Shape tool to draw over the dotted lines.

Select shapes (2/5)

Select the shapes you have drawn for the next step. Click the Selection tool and drag it across the shapes you drew.

Merge shapes (3/5)

Start merging shapes now. Use the Shape Builder tool (Shift + M) to paint over areas of the shape you want to merge.

Color your icon (4/5)

Color the shape to make the icon pop. Select your camera icon, and then click a color from the Swatches panel to color the icon.

Good to go! (5/5)

Woo hoo! Your camera icon is ready! Select the finished icon using the Selection tool (V), and click File > Export Selection.
